# Анализ активности студентов онлайн курса (учебный проект). 
Дана информация о действиях, которые совершают студенты со стэпами и данные о времени и статусах сабмитов к практическим заданиям.
Цель: обработать данные и предсказать на решающих деревьях, закончит ли студент курс по первым дням его активности.
