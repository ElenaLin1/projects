# Классификация типов рака кожи по клинико-демографическим признакам

## Описание проекта

Цель: на основе клинических и демографических признаков (возраст, пол, локализация, симптомы, факторы риска) предсказать, относится ли поражение кожи к одному из трёх злокачественных типов — BCC (базальноклеточная карцинома), MEL (меланома), SCC (плоскоклеточная карцинома) — или является доброкачественным.

Модель не заменяет биопсию, но может использоваться как инструмент поддержки врачебных решений для предварительной маршрутизации пациентов.

## Данные

Использован датасет PAD-UFES-20 (The PAD-UFES-20: a skin lesion dataset composed of patient data and clinical images collected from smartphones).

- Источник: [Kaggle](https://www.kaggle.com/datasets/mahdavi1202/skin-cancer)
- Количество записей: 1470 после очистки
- Признаки: возраст, пол, локализация, диаметр, симптомы (зуд, рост, боль, изменение, кровоточивость, возвышение), факторы риска (курение, алкоголь, пестициды, история рака), тип кожи по Фитцпатрику.

## Целевая переменная

Четыре класса:
- 0 — не рак (ACK, NEV, SEK)
- 1 — BCC
- 2 — MEL
- 3 — SCC

Сильный дисбаланс классов: BCC встречается значительно чаще, чем MEL и SCC.

## Методы

- **Предобработка:** удаление пропусков, удаление выбросов (нулевой диаметр), кодирование бинарных признаков (1/0), создание признака `sun_exposed` на основе локализации, удаление высококоррелированных признаков (diameter_2).
- **Разведочный анализ:** boxplot'ы возраста и диаметра по диагнозам, тепловая карта корреляций, тест Краскела-Уоллиса для числовых признаков, тест Хи-квадрат и V-мера Крамера для категориальных.
- **Моделирование:** разделение на train/val/test (60/20/20) со стратификацией. Опробованы:
  - Логистическая регрессия (базовая и с `class_weight='balanced'`)
  - Random Forest (базовый, с балансировкой, с подбором гиперпараметров через GridSearchCV)
  - LightGBM (с балансировкой и с подбором гиперпараметров)

## Результаты

**Финальная модель:** Random Forest с `class_weight='balanced'` без дополнительного подбора гиперпараметров.

**Выводы:**
- BCC определяется отлично (Recall 0.86).
- MEL определяется с высокой точностью (Precision 0.80), но низким Recall (0.40). Модель находит 40% реальных меланом и редко ошибается при их предсказании. Это значит, что при положительном прогнозе высока вероятность меланомы, но отрицательный прогноз не исключает её.
- Для улучшения необходимо больше данных по редким классам или применение SMOTE.
